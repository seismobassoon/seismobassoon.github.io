Seismobassoon — Hugo multilingual starter (minimal)
==================================================

This is a lightweight starter for a multilingual Hugo site for **Nobuaki Fuji**.

How to deploy from your iPhone (quick):
1. Create a new repository on GitHub named `seismobassoon.github.io`.
2. Open the repository, click "Add file" → "Upload files".
3. Upload the files and folders from this zip (extract on your phone or upload directly).
4. Commit. GitHub Pages will build the site automatically if you have a valid `config.toml`.
5. To use the Wowchemy theme later: add the theme as a submodule or replace the `theme` setting in `config.toml`.

Recommendations:
- Use the GitHub app or Safari to edit Markdown files directly.
- Add images in `static/images/` and reference them in pages.
- For BibTeX-powered publications, consider switching to the Wowchemy starter or Academic theme for full integration.

This starter intentionally keeps files small and editable on mobile. Customize the content in `content/<lang>`.
